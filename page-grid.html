A Complete Guide to CSS Grid

Source: A Complete Guide to CSS Grid

- Grid layout is intended for larger scale layouts.

- CSS Grid Layout, is a two-dimensional grid-based layout system

Parent Properties


    display					: grid | inline-grid;
    grid-template-columns	: ...  ...; //200px 200px //repeat(5, 1fr) // 50px auto 100px 1fr
                            : grid-template-columns: [first] 40px [line2] 50px [line3] auto [col4-start] 50px [five] 40px [end];
                            : repeat(3, 20px [col-start]);
                            : 1fr 1fr 1fr;
                            : 1fr 50px 1fr 1fr;
                            : minmax(100px, 1fr) 3fr;
                            : 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr;
                            : repeat(8, 1fr);
                            : repeat(8, minmax(10px, 1fr));
    grid-template-rows		: ... ...;
                            : [row1-start] 25% [row1-end] 100px [third-line] auto [last-line];
                            : [row1-start] 25% [row1-end row2-start] 25% [row2-end];
                            : masonry;
    grid-template-areas		: https://css-tricks.com/snippets/css/complete-guide-grid/#aa-grid-template-areas
    grid-template			: none | < grid-template-rows> / < grid-template-columns>;
    column-gap				: < line-size>;
    row-gap					: < line-size>;
    gap						: < grid-row-gap> < grid-column-gap>;
    justify-items			: start | end | center | stretch;
    justify-content			: start | end | center | stretch | space-around | space-between | space-evenly;
    align-items				: start | end | center | stretch;
    align-content			: start | end | center | stretch | space-around | space-between | space-evenly;
    * place-items			: center;
    grid-auto-flow			: row | column | row dense | column dense;
    * grid					: 100px 300px / 3fr 1fr;

Child Properties


    grid-column-start		: < number> | < name> | span < number> | span < name> | auto;
    grid-column-end			: < number> | < name> | span < number> | span < name> | auto;
    grid-row-start			: < number> | < name> | span < number> | span < name> | auto;
    grid-row-end			: < number> | < name> | span < number> | span < name> | auto;
    grid-column				: < start-line> / < end-line> | < start-line> / span < value>;
    grid-row				: < start-line> / < end-line> | < start-line> / span < value>;
    grid-area				: < name> | < row-start> / < column-start> / < row-end> / < column-end>;
    justify-self			: start | end | center | stretch;
    align-self				: start | end | center | stretch;
    place-self				: center stretch;

place-items

- place-items sets both the align-items and justify-items properties in a single declaration.

grid

- A shorthand for setting all of the following properties in a single declaration: 


    grid-template-rows
    grid-template-columns
    grid-template-areas
    grid-auto-rows
    grid-auto-columns
    grid-auto-flow
