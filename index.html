How to use @media in CSS

Source: A Complete Guide to CSS Media Queries 

    @media screen and (max-width: 576px) {
        /* styles will be apply if screen size 0 - 576*/
    }
    
    @media screen and (min-width: 577px) and (max-width: 992px) {
        /* styles will be apply if screen size 577 - 992*/
    }
